[[query-dsl-type-query]]
== Type Query

Filters documents matching the provided document / mapping type. Note,
this query can work even when the `_type` field is not indexed (using
the <<mapping-uid-field,_uid>> field).

[source,js]
--------------------------------------------------
{
    "type" : {
        "value" : "my_type"
    }
}    
--------------------------------------------------
